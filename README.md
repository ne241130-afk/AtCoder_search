# AtCoder Learning Hub

## 概要

AtCoderでアルゴリズム学習を行う学生向けのWebアプリケーションです。

AtCoder Problemsでは検索しづらい

- アルゴリズムタグ検索
- 問題管理
- 学習支援

を目的として開発しています。

---

## 開発目的

サークル活動では

「二分探索だけ練習したい」

「DPだけ解きたい」

という要望が多くありました。

しかしAtCoderではアルゴリズムごとに問題を検索することが難しいため、
タグ検索機能を備えた学習支援ツールを作成することにしました。

---

## 主な機能

- 問題検索
- タグ検索
- 問題詳細表示
- 難易度表示
- （今後追加予定）
  - AtCoderアカウント連携
  - 学習履歴管理
  - おすすめ問題表示
